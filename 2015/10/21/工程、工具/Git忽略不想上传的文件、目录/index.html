Git忽略不想上传的文件、目录

工程、工具 Git 🕒 2015年10月21日

项目开发工程中，总会有些不需要上传到仓库的文件，为了避免这些文件被误上传，git支持两种配置模式（配置方法完全相同），可以直接忽略配置的文件和目录。

  • 配置文件的名称：.gitignore
  • 配置忽略的内容：在.gitignore中添加想要忽略的文件或者目录，支持*等通配符方式。
